# Algoritmi-za-urejanje-podatkov
Seminarska naloga in predstavitev o algoritmih za urejanje podatkov (ang. sorting algorithms)

Zaključna seminarska naloga pri predmetu Matematične tehnologije, katerega izvajalec je dr. José Antonio Montero Aguilar.
Celotna seminarska naloga, skupaj s predstavitvijo je ustvarjena v `LaTeX`-u -- seminarska naloga kot `article`, predstavitev kot `beamer`.
V predstavitvi so uporabljene simulirane animacije, za katere so bile slike ustvarjene s programom Geogebra.

### Vsebina
Seminarska naloga obsega:
  * Osnove algoritmov za urejanje podatkov
  * *O*-notacijo
  * Primeri algoritmov za urejanje podatkov (*z razlago, primerom delovanja algoritma in časovno zahtevnostjo*)
    * Bubble sort
    * Insertion sort
    * Quick sort
    * Selection sort
    * Counting sort
    * Bogosort
  * Primerjava časovnih kompleksnosti
